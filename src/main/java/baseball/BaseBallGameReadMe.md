# 숫자야구게임 구현
## 기능
### 각각의 자리의 숫자가 겹치지 않도록 하는 3자리 숫자 생성
* 생성자를 통해 숫자 생성
  * 생성된 숫자를 Set에 삽입
  * 이미 존재하는 숫자인지를 검사하는 exitNumber 메서드
### 매개변수로 받은 숫자 검증
* Set에 포함되어있는 숫자라면
  * 자리에 맞는지 비교하는 validateStrike 메서드
### strike와 ball, nothing을 문자열로 보여주는 메서드 
* toString()을 오버라이드 하여 만들기
* 만약 strike와 ball 모두 0이라면 낫싱 출력
